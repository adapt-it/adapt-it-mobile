Translate between related languages on your mobile device.
Multiplatform
Work on a tablet or smartphone. Adapt It Mobile is compatible with both Apple and Android devices.
Simple, focused translation
Adapt It Mobile is designed specifically for mobile devices, with the same powerful adaptation engine found in the desktop version of Adapt It.
Work with text or Paratext files
Import .usx, .usfm, Adapt It .xml or plain-text files for adaptation. When you're done, you can export the files to your favorite editor for further revision and publication.
